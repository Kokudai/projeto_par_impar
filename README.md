## 📝 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de treinar lógica de programação em C#, aplicando validação de dados, estruturas condicionais, laços de repetição e interação com o usuário.  
A aplicação permite analisar números (par ou ímpar), gerar tabuadas, validar idade e altura, realizar um mini desafio lógico e aplicar um quiz com cálculo de nota e porcentagem.  
O foco principal é consolidar conceitos fundamentais de programação enquanto cria uma experiência divertida e interativa.

## 🚀 Funcionalidades

- Contagem de números de 0 até N indicando se cada um é par ou ímpar  
- Tabuada completa com indicação de par/ímpar  
- Validação de idade com limites reais  
- Validação de altura no formato correto  
- Teste interativo de par/ímpar  
- Quiz com 6 perguntas e nota final  
- Cálculo de porcentagem e feedback ao usuário
